Alignment


You are given two sentences, where some words and phrases have similar meanings. Your task is to align as many of them as possible.
Try to minimize the amount of tokens in each alignment, while keeping multi-word expressions intact. For example:

       obama visited canada yesterday                         
     
       the president flew to ottawa for a meeting with trudeau                         
    ✔ Obama   /   the president    
    ✔ Canada   /   Ottawa    
    ✔ visited   /   flew to    
    ✘ Obama   /   Trudeau    
    ✘ yesterday   /   for a meeting    
 

Selecting Alignments
  1. Select a token (or set of tokens) in the first sentence.
  2. Select a token (or set of tokens) in the second sentence, which has a similar meaning to the one selected in the first sentence.
  3. Press "Align".
  4. Repeat the process until all similar tokens have been aligned. Different alignments should have different colors.
  5. Click "Submit" when done.
Note: some tokens are "glued" to other tokens, and selecting them will automatically select others. That's fine.
